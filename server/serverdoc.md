# Server
Сервер запускается на домене http://localhost:3000 <br>
Для запуска нужно открыть вторую консоль в папке **server** <br>
и запустить его командой ***npm start*** <br>
В будущем проект вместе с сервером будет открыватся одной командой <br>

**!!!** Перед запуском необходимо помнить, что у сервера отдельные зависимости<br>
и их тоже нужно установить ***npm i***

## Страница index.html
### GET запрос
Для получения данных о команде нужно отправить GET запрос <br>
по роуту ***/team***, в ответе должен прийти массив объектов, <br>
 сам объект должен быть похожего вида:
```
{
    "id": 3,
    "fullName": "Николаева Наталия",
    "photo": "./img/team/Natasha.jpg",
    "dateOfBirth": "19.09.2001",
    "yo": 19,
    "height": 157,
    "weight": 47,
    "post": "Front-end developer",
    "hobby": [
      "психология",
      "танцы"
    ]
  }
```
### POST запрос
#### Этот пункт скороее всего поменяется!!!
Для редактирования данных команды необходимо отправить POST запрос <br>
на роут ***/team*** в body нужно передать массив объектов, с уже измененными данными.<br>
Поля id и photo **!!!НЕЛЬЗЯ!!!** изменять. Объект должен быть такого формата:
```
{
    "id": 3,
    "fullName": "Николаева Наталия",
    "photo": "./img/team/Natasha.jpg",
    "dateOfBirth": "19.09.2001",
    "yo": 19,
    "height": 157,
    "weight": 47,
    "post": "Front-end developer",
    "hobby": [
      "психология",
      "танцы"
    ]
  }
```
## Страница question.html
### GET запрос
Для получения вопросов необходимо отправить GET запрос <br>
на роут ***/questions***, так же необходимо указать в query параметрах <br>
тип файла с которого мы хотим прочитать данные. <br>
Пример как должена выглядеть строка для получения файлов из JSON: <br>
**http://localhost:3000/questions?file=json**   <br>
В ответе должен прийти массив объектов, сам объект должен быть такого типа: <br>
 ```
 {  
    "id" : 31234122
    "question": "Если все составные части исходного объекта были заменены, остаётся ли объект тем же объектом?",
    "answer": true,
    "theme": "philosophical",
    "formats": ['json','csv'] //в каких форматах записан вопрос
 }
 ```
### POST запрос
Для отправки данных необходимо сделать POST запрос по роуту  <br>
***/questions*** , а в body отправить объект вида:
```
 {  
    "id" : 31234122 // дата штамп
    "question": "Если все составные части исходного объекта были заменены, остаётся ли объект тем же объектом?", // собственно сам вопрос
    "answer": true, // ответ на вопрос
    "theme": "philosophical", // тема вопроса
    "formats": ['json','csv'] // в какие файлы хотите записать вопрос
 }
 ```
### DELETE запрос
Для удаления данных с сервера необходимо сделать DELETE запрос по роуту <br>
***/questions*** , а в body отправить объект вида:
 ```
 {  
    "id" : 31234122 // id удаляемого вопроса
    "format": "json" // из какого файла удалить
 }
 ```
